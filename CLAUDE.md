# CLAUDE.md

このファイルは、Claude Code がこのリポジトリで作業する際のルールを定めたものです。

## プロジェクト概要

このプロジェクトは **AI MONSTER**:業態を問わず使えるゲーム化アンケートLP(ランディングページ)のテンプレートシステムです。

## コミュニケーションのルール

- ユーザーはPC操作・プログラミングに不慣れです。**専門用語を使うときは、都度ひとことの説明を添える**こと(例:「CSS(ページの見た目を整える仕組み)」)。
- **作業前に必ず計画を提示し、ユーザーのOKを得てから実装する**こと。いきなりコードを書き始めない。
- **技術的な断言を避ける**こと。不確実な点・未検証の点は「要確認」と明示する。

## 表示確認のルール

- **スマホ縦画面(375px幅)での表示確認を常に最優先する**こと。PCでの見た目より先にモバイルを確認する。
- 横長グラフ・ワイド動画・横並び比較などのレイアウトは**縦画面で崩れやすい**。必ずモバイル時の構成(縦積みへの切り替えなど)まで確認すること。

## 実装上の禁止事項・注意点

- `mix-blend-mode: screen` は**白背景の上では機能しない**(白いまま何も見えなくなる)。**提案禁止**。
- 複数動画を同時再生する場合は、**表示サイズ相当に軽量化(CRF28前後・音声なし)してから配置する**こと。重い動画をそのまま並べない。
  - CRF:動画の圧縮率の設定値。数値が大きいほどファイルが軽くなる代わりに画質が下がる。
